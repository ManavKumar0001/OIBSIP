# Unemployment Rate Analysis During COVID-19

## Overview

This project analyzes unemployment trends using unemployment rate data collected across different regions. The objective is to understand the impact of COVID-19 on employment levels and identify patterns in unemployment rates through data visualization and statistical analysis.

The project focuses on exploring unemployment trends, regional variations, and changes over time using Python-based data analysis techniques.

---

## Objectives

* Analyze unemployment rate data.
* Identify trends before and during the COVID-19 period.
* Compare unemployment rates across regions.
* Visualize unemployment patterns using charts and graphs.
* Generate meaningful insights from real-world data.

---

## Dataset Information

The dataset contains information such as:

* Region
* Date
* Estimated Unemployment Rate (%)
* Estimated Employed Population
* Estimated Labour Participation Rate (%)

## Key Findings

* Significant increase in unemployment rates was observed during the COVID-19 period.
* Some regions experienced higher unemployment levels compared to others.
* Employment recovery trends became visible after restrictions were gradually lifted.
* Data visualization helped identify patterns and economic impacts more effectively.
