# Car Price Prediction Using Machine Learning

## Overview

Car prices are influenced by several factors such as brand reputation, vehicle age, fuel type, transmission, ownership history, and market demand. This project develops a machine learning model capable of predicting the selling price of a car using historical vehicle data.

The goal is to assist buyers and sellers in estimating a fair market value for a vehicle based on its characteristics.

---

## Problem Statement

Determining the correct selling price of a used car can be challenging due to multiple influencing factors. This project uses machine learning techniques to analyze historical car data and generate price predictions based on vehicle attributes.

---

## Dataset Features

The dataset contains information such as:

* Car Name
* Year
* Present Price
* Kilometers Driven
* Fuel Type
* Selling Type
* Transmission
* Owner Count
* Selling Price (Target Variable)

## Machine Learning Model

Linear Regression was used to learn the relationship between vehicle attributes and selling price.

The model predicts the estimated selling price based on the input features provided.

---

## Evaluation Metrics

The model performance was evaluated using:

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

These metrics help assess the accuracy and reliability of the predictions.
