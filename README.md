# OIBSIP – Oasis Infobyte Data Science Internship

## Overview

Welcome to my Oasis Infobyte Internship Project Repository. This repository contains the projects completed as part of the Oasis Infobyte Data Science Internship Program, where I worked on solving real-world data science and machine learning problems using Python and industry-standard libraries.

The internship provided an opportunity to strengthen my understanding of data analysis, data visualization, machine learning, model evaluation, and predictive analytics through hands-on project development.

Each project focuses on a different aspect of data science, ranging from classification and exploratory data analysis to regression-based predictive modeling.

## Projects Completed

### Task 1: Iris Flower Classification

#### Project Description

The Iris Flower Classification project focuses on building a machine learning model capable of identifying Iris flower species based on their physical characteristics.

The dataset contains measurements of sepal length, sepal width, petal length, and petal width for three species of Iris flowers:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

A classification model was trained to learn the relationship between these features and accurately predict the species of a flower.

#### Skills Demonstrated

* Data Loading and Exploration
* Data Visualization
* Classification Algorithms
* Model Training and Testing
* Performance Evaluation

#### Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

---

### Task 2: Unemployment Analysis During COVID-19

#### Project Description

The COVID-19 pandemic significantly impacted employment across different regions. This project focuses on analyzing unemployment trends using real-world unemployment datasets.

The analysis explores how unemployment rates changed over time and identifies patterns and regional variations that emerged during the pandemic period.

Various visualizations and statistical techniques were used to better understand the economic impact of COVID-19 on employment.

#### Skills Demonstrated

* Exploratory Data Analysis (EDA)
* Data Cleaning
* Statistical Analysis
* Trend Analysis
* Data Visualization

#### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

### Task 3: Car Price Prediction

#### Project Description

The price of a vehicle depends on several factors such as age, fuel type, transmission, ownership history, and market value. This project develops a machine learning model that predicts the selling price of a car based on historical vehicle data.

The project demonstrates how regression techniques can be used to estimate prices and support data-driven decision-making in the automobile market.

#### Skills Demonstrated

* Feature Engineering
* Data Preprocessing
* Regression Modeling
* Predictive Analytics
* Model Evaluation

#### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
